# openclaw-config

Agent configuration files for OpenClaw deployments managed by OpenClawPro.

## Files

- `IDENTITY.md` -- Agent identity and behavior rules. Defines the agent persona, security protocols, and operational boundaries.

## Usage

These files are installed to `~/.openclaw/agents/main/` during `openclawpro setup` or `openclawpro install-skills`. The OpenClaw gateway reads them to configure agent behavior.
